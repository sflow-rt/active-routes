About

SDN Active Route Manager monitors prefixes using BGP and traffic with sFlow. Active prefixes can be pushed to commodity switch hardware to create a high performance route cache.

Configuration

Edit the following startup settings and restart sFlow-RT.

ParameterDefaultDescription
bgp.startnoStarts BGP listener
bgp.port1179TCP port used to listen for BGP connections
arm.reflector.ip127.0.0.1IP address of route reflector
arm.reflector.as65000Local AS number to advertise to route reflector
arm.reflector.idLocal identifier (e.g. 0.0.0.1) to advertise to route reflector
arm.target.ipTarget router for top prefixes
arm.target.asLocal AS number to advertise to target router
arm.target.idLocal identifier (e.g. 0.0.0.2) to advertise to target router
arm.target.prefixes20000Number of prefixes to push to target router
arm.target.minvalue0Traffic threshold before pushing prefix
arm.sflow.ipsFlow source for traffic data (default is arm.reflector.ip)
arm.sflow.t20Traffic moving average duration in seconds
